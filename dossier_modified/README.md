# README

Bienvenue sur le Jeu du Vendredi ! Ce site a été développé par Louis Vanème (stage de seconde) entre le mercredi 17 et le vendredi 19 juin 2026.
## Lancer le site
Allez dans le dossier racine du projet. Puis, exécutez server.py (\dossier\server.py), puis, suivez les instructions dans la console. Ou bien rendez vous sur http://localhost:5000/ et profitez du Jeu du Vendredi !
## En cas de problème d'initialisation
En cas de problème pour initialiser le site, vérifiez l'arborescence du projet et la présence de tous les dossiers et leurs fichiers respectifs. Dans le cas où un des fichiers n'existe pas, retéléchargez le projet (si vous l'avez téléchargé). Si le fichier [[database.db]] est absent, rendez vous sur le fichier [[sqlite.py]], lancez le script, puis, réessayez d'initialiser l'application.
