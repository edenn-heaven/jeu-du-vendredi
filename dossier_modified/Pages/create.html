Importer un quiz depuis un fichier JSON

Importez un fichier .json respectant le format suivant pour charger un quiz complet d'un coup. Voir le format attendu ▼

Format JSON attendu : 
{
  "title": "Titre du quiz",
  "tags": ["tag1", "tag2"],
  "questions": [
    {
      "question": "Quelle est la capitale de la France ?",
      "type": "multiple",
      "answers": [
        { "text": "Paris",   "is_correct": true  },
        { "text": "Lyon",    "is_correct": false },
        { "text": "Marseille","is_correct": false },
        { "text": "Bordeaux","is_correct": false }
      ]
    },
    {
      "question": "La Terre est ronde.",
      "type": "truefalse",
      "answers": [
        { "text": "Vrai", "is_correct": true  },
        { "text": "Faux", "is_correct": false }
      ]
    }
  ]
}

• type : "multiple" ou "truefalse"
• answers : entre 2 et 6 réponses, au moins une avec "is_correct": true
• tags : optionnel, tableau de chaînes
